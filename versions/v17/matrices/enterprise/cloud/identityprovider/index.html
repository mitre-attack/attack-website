Identity Provider Matrix

Below are the tactics and techniques representing the MITRE ATT&CK® Identity Provider platform. The techniques below are known to target cloud-based identity-as-a-service (IDaaS) platforms such as Microsoft Entra ID and Okta. Identity providers are SaaS platforms that support identity management and single sign-on across multiple applications. The Matrix contains information for the Identity Provider platform.
