ipconfig

ipconfig is a Windows utility that can be used to find information about a system's TCP/IP, DNS, DHCP, and adapter configuration. [1]

ID: S0100
Associated Software: ipconfig.exe
Type: TOOL
Platforms: Windows
Version: 1.0

Last Modified: 17 October 2018
